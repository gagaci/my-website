# About

Morning Bot is a TypeScript Telegram bot that sends a daily personal briefing. It brings together weather, calendar events, Canvas assignments, and Jira focus items into short Telegram messages.

# Features

- **Daily schedule** — Runs on a configurable morning cron schedule
- **On-demand briefings** — Send `/brief` in Telegram to generate the briefing whenever you want
- **Multiple sources** — Combines weather, calendar, Canvas, and Jira context
- **Telegram-first** — Sends clean HTML-formatted Telegram messages

# Tech Stack

- TypeScript
- node-telegram-bot-api
- node-cron
- OpenAI
- Axios
